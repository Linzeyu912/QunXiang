4. 模块依赖架构

核心模块

core — 共享类型
llm — LLM 提供商抽象 (Ollama / Custom / Mock)
prompts — LLM 提示词模板
schemas — Zod 校验 schema

数据处理管道

import → preprocess → chapter-analysis
entity-prescan (正则+LLM+置信度+重要性)
story-arcs (故事弧线提取)

Agent 管道

extractors — LLM 批量提取
validators — 实体校验
entity-resolution — 消歧合并
scheduler — TaskDispatcher + EventBus
agent — ProducerAgent 主编排

存储 & API

storage — Prisma ORM (SQLite / PostgreSQL)
api — Fastify HTTP + JWT + SSE
web — React + Vite + Tailwind
exporters — JSON / CSV / Markdown 导出
